# HIVE-GAME
This project is a C++ implementation of the board game Hive, developed by five sophomores from the Sino-European School of Technology at Shanghai University. It faithfully recreates the original rules and offers two-player mode and single-player vs AI, with an optional extension piece – the Ladybug – to enrich strategic depth.

Core Mechanics: Six insect types (Queen, Ant, Spider, Beetle, Grasshopper, Ladybug) each follow unique movement rules on a hexagonal grid.

AI Opponent: Powered by a minimax algorithm to deliver challenging gameplay.

Graphical Interface: Built with Qt, featuring custom hexagon-shaped buttons for a clean and intuitive user experience.

Design Highlights: Implements design patterns such as Singleton, inheritance, and polymorphism; the code is well-structured with a complete UML class diagram for easy extension.
The project is open-source, Extract image.zip to drive D; click HIVE.pro to see all the code and run it，and a demo video is available on Bilibili: https://www.bilibili.com/video/BV1gHDrYqEZL
